## 📖 Overview

Web3 slang translator that explains crypto jargon to all newcomers with a humorous and blunt "veteran leek" persona. An experienced crypto trader who excels at using jokes to explain concepts and memes to break down complexity.

## ✨ Core Features

### 7-Dimensional Translation Framework

Every term includes complete outputs across these dimensions:

1. **Term Name** - Original English term (with full name if abbreviation)
2. **Plain Language Definition** - Most down-to-earth explanation possible
3. **Meme Origin Story** - Traces origin and cultural background
4. **Sentiment Analysis** - Bullish / Bearish / Neutral
5. **Scenario Simulation** - Real usage examples on X/Twitter, Discord
6. **Risk Level** - ⚠️⚠️⚠️⚠️⚠️ Risk rating
7. **Veteran Advice** - Specific, actionable practical tips

### Web Search Capability

- Auto-detects new terms (trending within 24 hours)
- Prioritizes X (Twitter) and Crypto media
- Real-time latest meanings and usage

### Classic Terms Library

Includes 15 classic Web3 term examples:
- HODL, FOMO, FUD, WAGMI, REKT
- DIAMOND HANDS, PAPER HANDS, RUG PULL
- DEGEN, APE IN, BAGHOLDER, WHALE
- And more...

## 🚀 Quick Start

### Platform Deployment

1. Download `web3-glossary-translator.skill` file
2. Upload Skill in Your Bot editor
3. Add Google Search or Bing Search plugin
4. Set preset questions as Start Message

### Claude Local Deployment

1. Clone repository:
```bash
git clone https://github.com/yourusername/web3-glossary-translator.git
```

2. Place `claude-web3-glossary-translator/` directory in Claude project

3. Claude automatically reads `CLAUDE.md` for guidance
